API

new Object()

Properties

id string
name string
x number
y number
layer number
vx number
vy number
rotation Vector
scale Vector
active boolean
childs Object[]
components Component[]

Object.id

Get identifier of Object

Do not modify this property.

Example 
/* Object identifier */
var o = new Object();
console.log(o.id);

Object.name

Get name of Object

Example 
var particleSystem = new Object();
particleSystem.name = 'Particle System';

Object.x

Get x position of Object

Example 
var player = new Object();
player.x++;

Object.y

Get y position of Object

Example 
var player = new Object();
player.y++;

Object.layer

Get layer (z-index) of Object

Example 
var menu = new Object();
menu.layer = 10;

Object.vx

Get x velocity of Object

Example 
var bullet = new Object();
bullet.vx = 8;

Object.vy

Get y velocity of Object

Example 
var bullet = new Object();
bullet.vy = 8;
