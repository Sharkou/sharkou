API

new Object()


Properties

Name Type
id string
name string
active boolean
x number
y number
layer number
rotation Vector
scale Vector
childs Object[]
components Component[]

Methods

Name Return
update() void
draw() void

Properties

Object.id

Get identifier of Object

Do not modify this property.

Example 
var o = new Object();
console.log(o.id); /* Object identifier */

Object.name

Get name of Object

Example 
var particleSystem = new Object();
particleSystem.name = 'Particle System';

Object.active

Active Object

Example 
var hud = new Object();
hud.active = false;

Object.x

Get x position of Object

Example 
var player = new Object();
player.x += 1;

Object.y

Get y position of Object

Example 
var player = new Object();
player.y += 1;

Object.layer

Get layer (z-index) of Object

Example 
var menu = new Object();
menu.layer = 10;

Methods

Object.update()

update is called every frame if Object is active

Example 
var player = new Object();
player.update = function() {
    if (player.life == 0) {
        player.active = false;
    }
};

Object.draw()

draw is called every frame if Object is active

Example 
var snow = new Object();
snow.x = 0;
snow.y = 0;
snow.draw = function() {
    circle(this.x, this.y);
};

Shapes


Functions

point()
line()
rect()
circle()

point(x, y)

Draws a point to the screen.


Parameter Description
x x position
y y position

Example 
var o = new Object();
o.draw = function() {
    point(20, 20);
    stroke('red');
    point(50, 80);
    stroke('white');
}

line(x1, y1, x2, y2)

Draws a line (a direct path between two points) to the screen.


Parameter Description
x1 the x-coordinate of the first point
y1 the y-coordinate of the first point
x2 the x-coordinate of the second point
y2 the y-coordinate of the second point

Example 
var o = new Object();
o.draw = function() {
    line(0, 0, 100, 100);
    stroke('blue');
}

rect(x, y, w, h)

Draws a rectangle to the screen.


Parameter Description
x x position
y y position
w width
h height

Example 
var o = new Object();
o.draw = function() {
    rect(50, 50, 50, 20);
    stroke('blue');
}

circle(x, y, r)

Draws a circle to the screen.


Parameter Description
x x position
y y position
r radius

Example 
var o = new Object();
o.draw = function() {
    circle(150, 50, 20);
    fill('green');
}

Color


Functions

fill()
stroke()
strokeWeight()

fill(color)

Sets the color used to fill shapes. RGB, RGBA and Hex CSS color strings and all named color strings are supported.


Parameter Description
color color

Example 
var o = new Object();
o.draw = function() {
    circle(50, 50, 30);
    fill('red');
}

stroke(color)

Sets the color used to draw lines and borders around shapes. RGB, RGBA and Hex CSS color strings and all named color strings are supported.


Parameter Description
color color

Example 
var o = new Object();
o.draw = function() {
    circle(50, 50, 30);
    stroke('red');
}

strokeWeight(weight)

Sets the width of the stroke used for lines, points, and the border around shapes. All widths are set in units of pixels.


Parameter Description
weight width (in pixels) of the stroke

Example 
var o = new Object();
o.draw = function() {
    circle(50, 50, 30);
    strokeWeight(5);
    stroke('red');
}

Keyboard


Attributes

keyPressed
keyReleased
key

keyPressed

The system variable keyPressed is true if any key is pressed and false if no keys are pressed.


Variable Type
keyPressed boolean

Example 
var o = new Object();
o.update = function() {
    if(keyPressed) {
        o.active = false;
    }
}

keyReleased

The system variable keyReleased is true if any key is released and false if no keys are pressed.


Variable Type
keyReleased boolean

Example 
var o = new Object();
o.update = function() {
    if(keyReleased) {
        o.active = true;
    }
}

key

The system variable key always contains the value of the most recent key on the keyboard that was typed.


Variable Type
key string

Example 
var o = new Object();
o.update = function() {
    if(keyPressed) {
        if(key === 'D') {
            o.translate(1, 0);
        }
    }
}

Mouse


Attributes

mousePressed
mouseReleased
mouseX
mouseY

mousePressed

The system variable mousePressed is true if the mouse is pressed and false if not.


Variable Type
mousePressed boolean

Example 
var o = new Object();
o.draw = function() {
    if(mousePressed) {
        circle(100, 100, 30);
        fill('#666');
    }
}

mouseReleased

The system variable mouseReleased is true if the mouse is released and false if not.


Variable Type
mouseReleased boolean

Example 
var o = new Object();
o.draw = function() {
    if(mouseReleased) {
        circle(100, 100, 30);
        fill('black');
    }
}

mouseX

The system variable mouseX always contains the current horizontal position of the mouse, relative to (0, 0) of the canvas.


Variable Type
mouseX number

Example 
var o = new Object();
o.draw = function() {
    line(mouseX, 0, mouseX, 100);
}

mouseY

The system variable mouseY always contains the current vertical position of the mouse, relative to (0, 0) of the canvas.


Variable Type
mouseY number

Example 
var o = new Object();
o.draw = function() {
    line(0, 0, mouseX, mouseY);
}
