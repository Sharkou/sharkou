Name Return

Properties

Object.id

Get identifier of Object

Do not modify this property.

Example 
var o = new Object();
console.log(o.id); /* Object identifier */

Object.name

Get name of Object

Example 
var particleSystem = new Object();
particleSystem.name = 'Particle System';

Object.active

Active Object

Example 
var hud = new Object();
hud.active = false;

Object.x

Get x position of Object

Example 
var player = new Object();
player.x += 1;

Object.y

Get y position of Object

Example 
var player = new Object();
player.y += 1;

Object.layer

Get layer (z-index) of Object

Example 
var menu = new Object();
menu.layer = 10;

Methods

Object.update()

update is called every frame if Object is active

Example 
var player = new Object();
player.update = function() {
    if (player.life == 0) {
        player.active = false;
    }
};

Object.draw()

draw is called every frame if Object is active

Example 
var snow = new Object();
snow.x = 0;
snow.y = 0;
snow.draw = function() {
    circle(this.x, this.y);
};

Functions

Name Return

point(x, y)

Draw a point

Parameter Description
x x position
y y position
Example 
point(10, 30);

line(x1, y1, x2, y2)

Draw a rectangle

Parameter Description
x1 the x-coordinate of the first point
y1 the y-coordinate of the first point
x2 the x-coordinate of the second point
y2 the y-coordinate of the second point
Example 
line(0, 0, 20, 20);

rect(x, y, w, h)

Draw a rectangle

Parameter Description
x x position
y y position
w width
h height
Example 
rect(30, 20, 55, 55);

circle(x, y, r)

Draw a circle

Parameter Description
x x position
y y position
r radius
Example 
circle(50, 50, 4);
